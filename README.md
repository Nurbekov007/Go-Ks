# CS2 Invitation Website 🎮

Сайт-приглашение для игры в Counter-Strike 2 в компьютерном клубе.

## Особенности:

### Главная страница (index.html):
- 🎬 Анимированный логотип VALVE при загрузке (как в CS2)
- 💬 Приглашение на казахском языке
- 🔵 Синяя кнопка "Согласен" (цвет CT - контр-террористов)
- 🟠 Оранжевая кнопка "Не Согласен" (цвет T - террористов)
- 😈 Прикольное сообщение при отказе

### Страница статистики (stats.html):
- 📊 Статистика 6 игроков в стиле Faceit
- 🌟 Особые карточки для MVP и загадочного игрока
- 📈 К/Д, попадания в голову, процент побед, прозвища
- 🎥 Ссылка на популярное видео с раскидками на Mirage
- 🎨 Темный дизайн в стиле CS2/Faceit

## Как загрузить на GitHub Pages:

1. Создайте новый репозиторий на GitHub
2. Назовите его, например: `cs2-invitation`
3. Загрузите файлы:
   - index.html
   - stats.html
4. В Settings → Pages → выберите `main` ветку и `/ (root)` папку
5. Сохраните и подождите 1-2 минуты
6. Ваш сайт будет доступен по адресу: `https://ваш-ник.github.io/cs2-invitation/`
